# 第1回 プロジェクト定例ミーティング議事録

日時: 2026-04-01 10:00〜11:00
参加者: 佐々木（PM）、小林（バックエンド）、松本（フロントエンド）、井上（インフラ）

## アジェンダ

* #2419 決済APIエラー増加問題
* #2425 管理画面CSV出力の文字化け
* #2421 本番サーバCPU使用率上昇

## 議論内容／決定事項

#2419 決済APIエラー増加問題
3月後半から決済失敗率が上昇している。ログ確認の結果、外部決済APIのタイムアウトが増加していることを確認した。当社側のタイムアウト値が短い可能性もあり調査を進める。議論の中でリトライ設計が未整備であることが判明し、別チケットで整理することに決定。→ #2420 決済APIリトライ設計見直し を新規作成（小林担当）。#2419は外部API影響の調査を継続する。

#2425 管理画面CSV出力の文字化け
Windows環境でCSVが文字化けする報告あり。UTF-8固定出力が原因と考えられる。BOM付きUTF-8で出力するよう修正対応を行う。松本が本日中に実装しステージングへ反映する。

#2421 本番サーバCPU使用率上昇
夜間バッチ実行時にCPU使用率が90%近くまで上昇。集計クエリが原因の可能性あり。実行計画を確認し、インデックス追加の必要性を検討する。井上が調査担当。
